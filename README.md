## Requirements 

- Swift 3.0
- iOS 8.0
- Xcode 8.0

## Installation 

WJClipsButton is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "WJClipsButton"
```
